All these library directories should be copied into your Arduino IDE libraries folder.
For 1.6.x versions an onwards this is in your sketches/ directory

PDQ_GFX Libraries.
Source: https://hackaday.io/project/6038-pdqgfx-optimzed-avr-lcd-graphics
I added some drawing functions (ellpises, et.al.)

DallasTemperature / DS18b20 Temperature Sensor (and OneWire)
Source: https://www.milesburton.com/Dallas_Temperature_Control_Library

Rotary Encoder:
Source: http://www.pjrc.com/teensy/td_libs_Encoder.html
I changed the direction of the encoder, so clockwise=+ anticlockwise=-

PID Library:
Source: http://playground.arduino.cc/Code/PIDLibrary
